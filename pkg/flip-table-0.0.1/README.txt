= flip-table

* http://philcrissman.com/gems/flip-table

== DESCRIPTION:

Sometimes, you just need to virtually flip a table over in frustration.

== FEATURES/PROBLEMS:

* Real-time enterprise ready scalable paradigm-disrupting table flipping
* Synergy
* UTF-8

== SYNOPSIS:

A command line tool for showing a little angry emoticon flipping over a table. For your cathartic viewing enjoyment.

Command line: run

  flip-table

And see:

  (ノಠ益ಠ)ノ彡┻━┻

== REQUIREMENTS:

I guess your terminal needs to be able to display utf-8 characters.

== INSTALL:

gem install flip-table
